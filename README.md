# Saylani Mass IT Hub Portal

React + Vite app scaffold for Saylani portal. Features:
- Firebase Authentication (Email/Password)
- Firestore collections for lost_found_items, complaints, volunteers, notifications
- Firebase Storage (placeholder ready)
- Realtime updates via onSnapshot
- Admin panel to change statuses

Setup:
1. Install dependencies: `npm install`
2. Add Firebase config in `src/firebase/firebaseConfig.js`
3. Run dev server: `npm run dev`

Notes:
- This scaffold includes core components, pages, and services. Replace Firebase config with your project settings.
